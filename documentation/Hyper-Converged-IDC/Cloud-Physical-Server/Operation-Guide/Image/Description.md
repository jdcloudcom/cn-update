# 镜像使用说明

## 功能描述

镜像（Image）是云物理服务器运行环境的模板，包含了“标准镜像”和“标准镜像+应用”两种模式。

### 标准镜像
目前云物理服务器的标准镜像支持：

|操作系统|
|:--:|
|CentOS-7.2  64位|
|CentOS-6.6 64位|
|Ubuntu-16.04 64位|
|Ubuntu-14.04 64位|

### 标准镜像+应用
目前云物理服务器的“标准镜像+应用”支持：

|操作系统|应用版本|
|:--:|:--|
|CentOS-7.2  64位|Nginx、MySQL、RabbitMQ|
|CentOS-6.6 64位|Nginx、MySQL、RabbitMQ|
|Ubuntu-16.04 64位|N/A|
|Ubuntu-14.04 64位|N/A|
