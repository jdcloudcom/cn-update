# 创建API分组

API 分组是 API 的基础管理单元。SDK是基于该API分组生成的。需先创建 API 分组，然后在分组下创建 API。
- 分组具有 Region 属性，API 选定分组前需确定 Region，且一旦生成不可更改。
- 基于API分组，用户若需发布服务还需配置基于分组的流控策略、授权访问、后端签名、自定义域名。
